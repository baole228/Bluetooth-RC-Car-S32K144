# Bluetooth RC Car — NXP S32K144

## Overview

Bluetooth-controlled RC car developed using the NXP S32K144 microcontroller.

## Features

- Bluetooth control via HC-05
- DC motor PWM control using FTM
- Servo steering control
- Independent steering and throttle
- UART communication
- L298N motor driver
- Speed control
- Forward / Reverse / Left / Right

## Hardware

- NXP S32K144
- HC-05
- L298N
- SG90 Servo
- DC Motor
- 3×18650 Battery

## Software

- S32 Design Studio
- Embedded C
- SDK RTM 4.0
